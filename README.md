# ⚽ World Cup Timer

Interactive countdown and broadcast timer inspired by FIFA World Cup TV graphics.

## Features

- Countdown to kickoff
- Team selection
- Country flags
- Match stage selection
- Full-screen broadcast mode
- Responsive layout

## Technologies

- HTML5
- CSS3
- JavaScript
